# @willet/shared

Core library for Willet — types, database, embeddings, and MCP tool definitions.

This package is used internally by [@willet/mcp](https://www.npmjs.com/package/@willet/mcp) and [@willet/server](https://github.com/SeriousBug/willet/tree/main/packages/server). You probably want to install `@willet/mcp` instead.

See the [Willet repository](https://github.com/SeriousBug/willet) for documentation and usage instructions.
