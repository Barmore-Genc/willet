# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Willet, please **do not** open a public GitHub issue. Public disclosure of vulnerabilities puts all users at risk.

Instead, please report it privately by emailing **willet.support@bgenc.dev**. Include as much detail as you can: steps to reproduce, affected versions, and potential impact.

## What to Expect

- We aim to acknowledge your report within **7 days**.
- We will work with you to understand the issue, develop a fix, and coordinate disclosure timing.
- We will credit reporters in the release notes unless you prefer to remain anonymous.

Thank you for helping keep Willet and its users safe.
